# 슬라이드 재생성 프롬프트

전체 에피소드를 다시 만들지 말고 실패한 카드만 재생성합니다.

```text
Regenerate only card {card_index}.

Keep:
- same character identity
- same story role
- same headline meaning
- same visual style

Fix:
{problem_to_fix}

Previous issue:
{previous_issue}

Updated direction:
{updated_direction}

Do not change:
{must_not_change}
```

## 자주 쓰는 수정 지시

- 배경을 더 단순하게 만든다.
- 캐릭터 얼굴을 레퍼런스와 더 비슷하게 만든다.
- 말풍선 수를 줄인다.
- 텍스트를 더 크게 배치한다.
- 표정보다 행동으로 감정을 보여준다.
- 1080 x 1080 안전 영역 안으로 주요 요소를 옮긴다.
