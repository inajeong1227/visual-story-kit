# Example Character: Mina

Mina는 Visual Story Kit의 예시 캐릭터입니다.
특정 주제에 묶이지 않고, 인스타툰, 카드형 설명 콘텐츠, 브랜드 스토리 예제에 사용할 수 있는 범용 주인공입니다.

## 1. 기본 정보

- 이름: Mina
- 역할: 주연
- 나이대: 20대 후반-30대 초반
- 성격 키워드: 관찰형, 약간 걱정 많음, 실행력 있음
- 콘텐츠 역할: 독자가 겪는 문제를 대신 발견하고 정리하는 화자

## 2. 고정 외형

- 얼굴형: 둥근 얼굴, 부드러운 턱선
- 눈 모양: 작고 둥근 눈, 검은 눈동자
- 헤어스타일: 턱선 길이의 짧은 단발, 옆 가르마
- 체형: 평균 체형, 머리 3.5-4등신의 단순 캐릭터 비율
- 기본 의상: 라벤더 셔츠, 진회색 바지
- 대표 소품: 작은 노트북 가방, 검은색 스마트폰

## 3. 컬러 팔레트

- 머리색: `#3A2F2A`
- 피부톤: `#F4C7A8`
- 눈 색: `#222222`
- 기본 의상 메인: `#B8A7E8`
- 기본 의상 서브: `#4A4A4A`
- 대표 소품: `#2E2E2E`

## 4. 스타일 고정값

- 선 스타일: 단순하고 둥근 손그림 선
- 선 두께: 중간 두께, 외곽선은 살짝 더 굵게
- 명암 수준: 최소 명암, 거의 플랫 컬러
- 배경 밀도: 매우 단순, 캐릭터보다 배경이 튀지 않음

## 6. 반드시 유지할 것

- 짧은 단발과 옆 가르마
- 둥근 얼굴과 작은 둥근 눈
- 라벤더 셔츠
- 3.5-4등신의 단순 캐릭터 비율
- 작은 노트북 가방 또는 스마트폰 중 하나

## 7. 바뀌어도 되는 것

- 표정
- 손동작
- 서기, 걷기, 앉기 같은 자세
- 장면에 맞는 임시 소품

## 8. 반드시 피할 것

- 긴 머리로 바뀌는 것
- 실사풍 또는 과한 애니메이션풍
- 큰 반짝이 눈
- 매 카드마다 다른 옷으로 바뀌는 것
- 레퍼런스 이미지의 배경 분위기를 따라 하는 것
