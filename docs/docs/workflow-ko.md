# 전체 제작 흐름

## 1. Brief

콘텐츠의 목적을 정합니다.

- 누구에게 보여줄 것인가
- 어떤 감정을 남길 것인가
- 몇 장으로 만들 것인가
- 정보형, 공감형, 에세이형, 브랜드형 중 무엇인가

## 2. Story Plan

이야기를 시작, 전개, 마무리로 나눕니다.

```text
오해 또는 질문
-> 상황 설명
-> 감정 또는 문제
-> 반전/정보
-> 내 관점
-> 결론/질문
```

## 3. Card Planning

각 카드는 아래 필드를 가집니다.

- headline
- scene
- narration
- dialogue
- caption
- visual_prompt
- regeneration_note

## 4. Character Lock

캐릭터가 흔들리지 않도록 변하지 않는 요소를 분리합니다.

- 얼굴형
- 눈 모양
- 헤어스타일
- 체형
- 기본 의상
- 대표 소품
- 선 스타일
- 색감

## 5. Prompt Generation

카드별 프롬프트는 장면 설명과 캐릭터 고정값을 합쳐 만듭니다.

## 6. Image Review

- 캐릭터가 같은 사람처럼 보이는가
- 텍스트가 읽히는가
- 1080 x 1080 안전 영역 안에 핵심 요소가 있는가
- 한 컷에 메시지가 너무 많지 않은가
- 다음 카드로 넘기고 싶게 만드는가

## 7. Regeneration

실패한 카드는 전체를 다시 만들지 말고 해당 카드만 재생성합니다.

## 8. Export

최종 이미지는 캐러셀 순서, 썸네일, 캡션, CTA까지 함께 정리합니다.
