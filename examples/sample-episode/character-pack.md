# Sample Episode Character Pack

이 에피소드는 `characters/example-character/profile.md`의 Mina를 사용합니다.

## Character Reference

```text
Character pack: characters/example-character/profile.md
Reference folder: characters/example-character/references/
```

## Character Lock Summary

- Name: Mina
- Role: main narrator
- Face: round face, small round eyes
- Hair: short bob, side part
- Outfit: lavender shirt, dark gray pants
- Props: small laptop bag, black smartphone
- Style: simple Korean instatoon, rounded hand-drawn line, flat colors

## Must Keep

- same short bob and side part
- same round face and small eyes
- same lavender shirt
- same simple 3.5-4 head character proportion
- same calm, observational narrator energy

## Must Avoid

- realistic rendering
- long hair
- large anime eyes
- new outfit per card
- copying reference background, lighting, or mood

## Reference Prompt Rule

Use this rule in every card image prompt:

```text
Extract only the character identity from reference images.
Ignore background, lighting, mood, and environment unless explicitly requested.
```
